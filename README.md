# ViewLingo Cam

Real-time camera translation app for iOS with AR overlay.

## Features

- **Live Camera Translation**: Point your camera at any text for instant translation
- **AR Overlay**: Translations appear naturally over the original text
- **On-Device Processing**: 100% offline using Apple Translation Framework
- **Privacy First**: No data leaves your device

## Requirements

- iOS 18.0+
- iPhone 12 or later (recommended)

## Tech Stack

- Swift 6.0
- Vision Framework (OCR)
- Translation Framework (On-device translation)
- AVFoundation (Camera)
- UIKit

## Installation

1. Clone the repository
2. Open `ViewLingo-Cam.xcodeproj` in Xcode
3. Build and run on your device
